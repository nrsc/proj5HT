QC and UMAP of patch-seq cells

Jeremy Miller

March 26, 2025

Overview

This code describes how to generate QC calls and UMAP coordinates for the patch-seq cells using Seurat.

This must be run after the first two code documents were run; however, this code block was run on a Windows machine since the docker environment does not work well with Seurat. See sessionInfo() at the end for details. We also assume you have already moved to an appropriate working environment.

Open the preprocessed and premapped data and set-up the space

Now lets prepare our environment.

Integrate patch-seq and and FACS cells using Seurat FOR QC!

In this section we are going to integrate Patch-seq and dissociated cells using Seurat. First, lets set up the Seurat object.

## Create data and metadata files
allGenes     <- intersect(rownames(datPatch),rownames(datMTG))
all.data     <- cbind(datMTG[allGenes,],datPatch[allGenes,])

                            subclass  = c(as.character(annoMTG$subclass_label),subclassPatch),
